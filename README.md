Bước 1: Clone project về
Bước 2: Trên terminal nhập: cd path_to_yolov10_folder
Bước 3: Trên terminal nhập:

Bước 5: Nhập dòng lệnh xuất hiện ở terminal streamlit run Helmet_Safety_Detection.py
